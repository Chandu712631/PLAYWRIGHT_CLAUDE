# Playwright Claude Project

This repository contains a Playwright test project.

## Scripts

- `npm test` — run Playwright tests
- `npm run test:headed` — run tests with headed browser
- `npm run test:report` — open the HTML test report
- `npm run codegen` — start Playwright code generator

## Setup

Install dependencies:

```bash
npm install
```

Run tests:

```bash
npm test
```
